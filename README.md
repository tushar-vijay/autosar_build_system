*AUTOSAR BUILD SYSTEM*
This is a initial delivery project with dummy .c and .h files to showcase build system of the project. Following are the folders present in the project:
- asw
- bsw
- cdd
- mcal
Each of these folders contain their own .c and .h files. They also contain CMakeLists.txt file for implementing CMake.

**Technologies used**
- C Programming Language
- CMake
- Batchfile Scripting
